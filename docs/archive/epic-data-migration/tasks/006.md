# Task 006: Hook scripts: on-agent-stop.sh and on-bash-complete.sh DB migration

## Description
Rewrite both hook scripts to query `epic_dispatch_chains` via `yoke-db.sh epic` instead of globbing `yoke/epics/*/status/dispatch-*.json` files. Update `find_project_root()` to validate `yoke/yoke.db` existence instead of `yoke/epics/`.

**on-agent-stop.sh changes:**
1. Replace the `for chain in "$PROJECT_ROOT"/yoke/epics/*/status/dispatch-*.json` loop with a DB query via `yoke-db.sh epic dispatch-chain-list` or direct SQL.
2. For each active chain, read `current_task` and `worktree_path` from DB.
3. Read task status from `epic_tasks` table instead of status JSON file.
4. Call `update-status.sh` with the DB-native interface (epic-slug, task-num) instead of the file path.
5. Update `find_project_root()` to check for `yoke/yoke.db` instead of `yoke/epics`.

**on-bash-complete.sh changes:**
1. Same dispatch chain discovery migration as on-agent-stop.sh.
2. Update `sync-progress.sh` call to pass DB-based arguments.
3. Update `find_project_root()` to check for `yoke/yoke.db`.
