# Yoke 0.1.1+launch.8

This release makes universe archives portable across the known schema changes
shipped before launch.8 while keeping the restore boundary fail-closed.

- Older archives restore into the current trusted schema without executing
  uploaded DDL.
- Project onboarding runs and checklist history are preserved and their tables
  now converge on every universe.
- Historical column order, the QA artifact-handle rename, and the nullable
  GitHub sync receipt fields are bridged losslessly.
- Unknown tables, columns, sequences, and executable restore syntax remain
  refused.

The normal installer and upgrade paths require no manual migration. Hosted
universe uploads from older releases require the target tenant to run launch.8
or later.
