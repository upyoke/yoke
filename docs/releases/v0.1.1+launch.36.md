# Yoke 0.1.1+launch.36

This release completes the universe workbench read surface. Strategy, Items,
Events, and Projects now render live data; item rows open a typed detail view,
epics include their tasks, and hosts may name the actor currently operating the
universe without inventing an identity in local mode.

The browser runtime keeps its shell, navigation, and view renderers in separate
closed-roster modules under the authored-file limit. A shared header-frame
asset now keeps the packaged universe UI and generated webapp template aligned.

The public installer is also the sole documented install and upgrade path, so
users retain its lockstep version resolution and protected package-index
configuration instead of copying an ambiguous multi-index uv command.
