# Yoke 0.1.1+launch.37

This release completes the shared universe read screens with Ouroboros. The
local, self-hosted, and hosted universe app now calls the registered,
project-scoped observation read and renders when, category, agent, context, and
review state without adding a mode-specific API.

The server allowlist and browser behavior test cover the same shared path, so a
future bundle cannot silently return Ouroboros to a placeholder screen.
