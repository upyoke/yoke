# Yoke 0.1.1+launch.23

This release makes the complete Yoke skill router portable to managed project
checkouts.

- Project installation now writes the canonical `.agents/skills/yoke` tree in
  addition to the Claude and Codex discovery copies, so every root-router and
  phase handoff resolves outside the Yoke source repository.
- The three installed skill trees are generated from one source and verified
  byte-for-byte, including a guard that rejects references to missing skill
  files.
- Retired scripts-directory guidance has been removed from the shipped
  Advance and Conduct flows, preventing managed-project agents from following
  source-only paths that no longer exist.

Use `yoke project install` to refresh an existing managed checkout. No schema
migration is required.
