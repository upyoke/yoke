# Yoke 0.1.1+launch.40

## Highlights

- Brings the universe workspace, navigation, tabs, panels, pills, and glyphs
  into the current visual system.
- Adds route-addressed Delivery views backed by live deployment-run history.
- Replaces the Sessions placeholder with a live roster showing session
  liveness and held work claims.
- Keeps hosted slot content in the normal page flow so narrow layouts no
  longer squeeze or overlap the main text column.

## Upgrade notes

Use the normal update path; hosted environments must advance their pinned Yoke
release to `0.1.1+launch.40` to receive these interface changes.

## Compatibility

None.
