# Yoke 0.1.1+launch.35

This release carries the managed-runner GitHub authority separation from
launch.34 together with the expanded universe workbench navigation. The
workbench now exposes stable destinations for the full product surface,
remembers project scope per view, and keeps hosted account screens in the
host application's outer navigation.

The browser runtime serves the extracted navigation module through the same
closed packaged-asset contract as the rest of the universe UI.
