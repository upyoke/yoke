# Yoke 0.1.1+launch.16

This release converges hosted universes onto one canonical physical schema and
makes operator database access follow managed credential rotation.

- A governed, content-preserving migration normalizes the function-call ledger
  layout so universe schema fingerprints agree across environments.
- Database-admin profiles resolve the current AWS-managed database credential
  at connection time instead of retaining a copied password.
- Authentication recovery refreshes the current credential after a password
  rejection and uses the previous managed version only during the narrow
  rotation window.
