# Yoke 0.1.1+launch.34

This release separates customer-facing GitHub repository authority from the
privileged GitHub App used by managed CI runners. Runner fleets can select an
explicit verified repository capability, so the Product App no longer needs
Administration or Webhooks permissions.

It also adds an audited machine-capability bootstrap mode for runner-fleet
recovery when the hosted token broker cannot repair itself, and makes the
manual skip-hop health-check timing test deterministic under xdist.
