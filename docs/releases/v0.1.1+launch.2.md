# Yoke 0.1.1+launch.2

## Highlights

- The public installer now runs its helper in an isolated uv environment, so
  an active or checkout-local virtual environment cannot substitute stale Yoke
  packages during installation verification.
- The installer resolves uv's authoritative tool-shim directory, including
  XDG layouts where the bin directory is not `~/.local/bin`, before smoking the
  installed command or launching onboarding.

## Upgrade notes

Rerun the normal installer. It force-reinstalls the exact stable product package
set and verifies that every Yoke package reports the channel version.

## Compatibility

None.
