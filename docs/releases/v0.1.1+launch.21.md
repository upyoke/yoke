# Yoke 0.1.1+launch.21

This release hardens interactive onboarding across normal and compatibility
terminal renderers.

- The onboarding wizard now handles both scrolling and plain-glyph body
  containers when restoring focus or forwarding the first typed character.
- Enter, Tab, Escape, and other control keys remain owned by their widgets
  instead of entering the printable-text handoff path.
- Public-repository URL submission no longer crashes on plain-glyph terminals,
  including the packaged macOS onboarding path.

Use the normal update path. No schema migration is required.
