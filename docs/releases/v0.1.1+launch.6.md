# Yoke 0.1.1+launch.6

This release hardens the public workbench and reusable hosted-infrastructure
template after the launch5 hosted onboarding cut.

- Capability actions now run during the originating browser event, preserving
  the transient user activation required by native file pickers and similar
  trusted browser surfaces.
- Rendered VPS stacks now accept an optional IAM instance profile, so hosted
  services can use capability-scoped cloud authority without copying static
  AWS credentials onto the server.
- The mount contract verifies synchronous invocation for both button and
  option-based capability actions, and renderer coverage verifies the
  instance-profile setting from project configuration through Pulumi output.
