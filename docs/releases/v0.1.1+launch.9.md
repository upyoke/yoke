# Yoke 0.1.1+launch.9

This release completes the GitHub-connected onboarding flow for public and
private repositories while preserving the launch.8 universe portability
contract.

- The onboarding wizard restores an explicit GitHub connection summary and
  confirmation step before repository selection.
- Private repository selection no longer falls back to raw clone URLs when
  GitHub access is unavailable or an error occurs.
- Empty private-repository access states provide clear manage, retry, and back
  actions.
- Repository listing paginates within one bounded request deadline so larger
  installations remain responsive.
- Clone-source settings and checklist state stay aligned with the selected
  GitHub repository throughout onboarding.

Hosted migration targets remain compatible with older universe archives, and
the packaged installer now includes the complete GitHub onboarding experience.
