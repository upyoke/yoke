# Yoke 0.1.1+launch.39

This release exposes project-owned deployment environment settings through the
registered function API. Local, self-hosted, and hosted operators can read and
CAS-merge the same settings document without reaching for a direct database or
a transport-specific command.

The server verifies that each environment belongs to the named project before
reading or writing it. Agent bootstrap packets and the packaged install bundle
teach the hosted-safe command and distinguish it from the retained local-only
domain helper.
