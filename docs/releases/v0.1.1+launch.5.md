# Yoke 0.1.1+launch.5

This release completes hosted GitHub onboarding and ordinary hosted machine
sign-in.

- `yoke connect` and the onboarding wizard now use browser approval to select
  one hosted organization and receive its scoped connection without token
  paste.
- Hosted project GitHub status now carries durable timestamps and outcomes from
  real installation-token automation.
- Product GitHub App operations guidance now documents the live hosted callback,
  setup, webhook, secret-custody, and automatic lifecycle-event behavior.
- Explicit token-file/stdin connection remains available for self-hosted and
  noninteractive operator use.
