# Yoke 0.1.1+launch.32

This release makes bounded source-authority receipt readers safe on PostgreSQL
autocommit connections. Named cursors now run inside explicit transaction
scopes, including full table digests and the project-capability and
capability-secret receipt planes used during hosted tenant replacement.

The change preserves bounded client memory and remains safe inside an existing
snapshot transaction. A real PostgreSQL regression covers the autocommit path.
