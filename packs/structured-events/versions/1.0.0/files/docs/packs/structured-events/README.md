# Structured Events Pack

Provides matching Python and TypeScript event envelopes, attribution helpers,
an API example, and contract tests.

## Project-specific work

- Set the project identity and environment at runtime.
- Choose the durable event transport, retention, privacy, and sampling policy.
- Extend the event vocabulary without placing user secrets in envelopes.
- Wire attribution to the project's real authentication and request context.
- Run the included cross-language contract tests after customization.
